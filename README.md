# Fitness Recommendation System

A web-based fitness recommendation application built using Django and Pandas that provides personalized exercise recommendations based on user preferences.

## Features
- Personalized exercise recommendations
- Goal-based filtering
- Difficulty-level selection
- Workout location filtering
- Dynamic recommendation system using CSV datasets

## Technologies Used
- Python
- Django
- Pandas
- HTML/CSS
- CSV Dataset

## Project Highlights
- Developed a fitness recommendation web application using Django and Pandas
- Built personalized exercise recommendation logic based on fitness goals, difficulty, and workout location
- Designed frontend forms and backend data-processing pipeline for dynamic recommendations
- Integrated CSV-based datasets with filtering and recommendation architecture
- Implemented modular recommendation engine using object-oriented Python design
